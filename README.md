# Python 3

Repositorio personal de **notas, ejemplos y ejercicios prácticos en [Python 3](https://www.python.org/)**.

El objetivo principal de este repositorio es servir como:
- **Referencia rápida** para repasar conceptos.
- **Apoyo para resolver dudas** comunes en Python.
- **Recordatorio práctico** de sintaxis, patrones y buenas prácticas.

No pretende ser un curso completo ni una guía oficial, sino un conjunto de apuntes organizados que pueden resultar útiles tanto para mí como para cualquier persona que esté iniciando o reforzando conocimientos en Python.

## Estructura del Proyecto

- Archivos principales (01-07): conceptos base y fundamentales
- Archivos complementarios (08-09): temas de apoyo
- Carpeta `utils/`: utilidades y funciones auxiliares
- Carpeta `static/`: archivos de texto y logs
- Carpeta `img/`: imágenes de apoyo

## Temario

1. **Fundamentos de Python** (`01_fundamentos.py`)
2. **Estructuras de Control** (`02_estructuras_control.py`)
3. **Funciones** (`03_funciones.py`)
4. **Manejo de Excepciones** (`04_excepciones.py`)
5. **Entrada y Salida de Datos** (`05_entrada_salida.py`)
6. **Empaquetado de Código** (`06_empaquetado.py`)
7. **Programación Orientada a Objetos (POO)** (`07_POO.py`)
8. **Logging** (`08_logging.py`)
9. **Bases de Datos** (`09_base_datos.py`)
